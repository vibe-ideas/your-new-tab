# Privacy Policy

Your New Tab replaces the browser new tab page with a configurable clock, search box, shortcuts, and background experience.

## Data Stored Locally

The extension stores configuration in the browser on the user's device:

- Bookmark source settings and cached bookmark data
- Search provider settings and the last selected provider
- Search history entered in the new tab search box
- Custom animated background URLs and the selected background index
- Language preference
- Cached default background image data

This data is used only to render and configure the new tab page.

## Data Sent to Third Parties

The extension does not send data to the developer's servers.

Some features contact third-party services directly from the user's browser:

- Running a search opens the selected search provider with the user's query.
- The default static background feature requests images from Unsplash or Picsum.
- If the user configures a remote bookmark JSON URL or custom animated background URL, the browser requests those URLs to load the configured content.

Those third-party services may receive standard request metadata such as IP address, user agent, and the requested URL according to their own policies.

## Remote Code

The extension does not fetch or execute remote code. Remote bookmark and background content is treated as data for display.

## Data Sharing

The extension does not sell, transfer, or share user data with the developer or third parties.

## Data Removal

Users can remove stored extension data by resetting the extension configuration, clearing browser site data for the extension, or uninstalling the extension.
